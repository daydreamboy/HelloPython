### （1）包和模块的关系[^1]

模块（module）是单个文件，而包（package）是一组模块的集合。它们的引用方式也不一样，如下

* 导入模块

```python
import my_module
```

* 导入函数

```python
from my_package.timing.danger.internets import function_of_love
```
